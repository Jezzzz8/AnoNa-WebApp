AnoNa

AnoNa is a fast, anonymous, link-based web app for group decisions and random draws. No login. No signup. Just create, share, and decide.

Built for Filipino group chats where people often ask: “Ano na?”

---

Features

PollNa 🗳️

A real-time voting tool for group decisions.

* Create polls in seconds
* Add custom options
* Set a timer
* Share link or QR code
* Live results update
* One vote per device

Result states:

* SolbNa: final decision
* LabanNa: tie
* WalaPa: no votes yet

---

PullNa 🎁

A random name draw tool for exchange gifts and assignments.

* Add participant names
* Generate secure random assignments
* Share link or QR code
* Each person reveals their own result only
* One reveal per device
* No duplicates or self-assignments

Status states:

* PullNa: assignment ready
* NakuhaNa: recipient revealed
* WalaPa: not started

---

How it works

1. Create a PollNa or PullNa event
2. Share the link or QR code
3. Friends join instantly
4. Vote or draw
5. See results or private assignments

---

Tech Stack

* React + Vite
* Supabase or Firebase (Realtime DB)
* React Router
* Recharts
* QR Code generator

---

Core Idea

Fast group decisions.

No accounts. No friction. Just results.

AnoNa → PollNa → PullNa → SolbNa 🚀
